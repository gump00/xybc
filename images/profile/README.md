# 图片放置说明

## 个人照片 Profile Photos
- hero-photo.jpg (600x600px) - 首页大头像
- about-photo.jpg (500x500px) - 关于页面头像

## 使用说明
1. 将个人专业形象照片重命名为上述文件名
2. 放置在此目录中
3. 图片会自动加载到网站对应位置
4. 如果图片不存在，会显示占位符文字

## 推荐规格
- 格式: JPG/PNG
- 背景: 简洁或去背景
- 构图: 正方形或圆形构图
- 质量: 高清晰度，文件大小控制在500KB以内

## 注意事项
- 确保图片文件名与代码中的引用完全一致
- 建议使用专业形象照片
- 可以使用相同照片但调整尺寸适配不同位置
