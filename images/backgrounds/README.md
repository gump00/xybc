# 背景图片说明

## 用途
此目录用于存放网站背景图片，可以替换或叠加现有的CSS渐变背景

## 推荐规格
- 尺寸: 1920x1080px 或更高
- 格式: JPG (文件较小) 或 PNG
- 大小: 建议控制在1MB以内
- 风格: 与紫色主题协调

## 使用场景
- hero-bg.jpg - 首页背景图
- section-bg.jpg - 各部分背景图
- texture-overlay.png - 纹理叠加图片

## 注意事项
- 确保图片不会影响文字可读性
- 可以使用半透明叠加来调整对比度
- 建议准备多个尺寸适配不同设备

## 实施方法
在CSS中通过background-image属性引用这些图片
